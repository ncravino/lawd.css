The Chonky Scale

You can adjust the default text size(1.1rem) by overriding the --default-chonk-size variable

.tiny will remove 0.3rem from --default-chonk-size

.thin will remove 0.2rem from --default-chonk-size

.fine will remove 0.1rem from --default-chonk-size

.chonk will be equal to --default-chonk-size

.heckinchonker will add 0.1rem to --default-chonk-size, h6 will also be set to this size.

.heftychonk will add 0.2rem to --default-chonk-size, h5 will also be set to this size.

.megachonk will add 0.3rem to --default-chonk-size, h4 will also be set to this size.

.gigachonk will add 0.4rem to --default-chonk-size, h3 will also be set to this size.

.terachonk will add 0.5rem to --default-chonk-size, h2 will also be set to this size.

.ohlawd will add 0.6rem to --default-chonk-size, h1 will also be set to this size.

Notouriousness

Here we can add a border around each letter to make it stand against backgrounds that have a mix of light and dark

You can change the color overriding --default-notoriousness-colour which is rgba(128,128,128,0.5) by default.

.notorious will add a 1px border using shadow around each letter

.blatant will add a 1.5px border using shadow around each letter
